## 💡 Usage Instructions

### Automatic Tracking
The Skills Tracker is **automatically updated** when you use these commands:
- `/analyse` - Detects concepts used in your solution
- `/update` - Confirms and records concepts in tracker
- `/summarise` - Reviews concepts learned in current problem

### Manual Tracking (if needed)
If you want to manually add a concept:
```
Concept: [Name]
Category: [Data Structure/Algorithm/Pattern/Optimization]
First Used: [Date] - [Problem Name (#Number)]
Usage History: [Problem1 (Date)], [Problem2 (Date)], ...
```

### View Skills Status
Use `/current` command to see:
- Total concepts mastered
- Most frequently used concepts
- Recent skill additions
- Mastery distribution
